get_field/2

get_field(Key::string(), Args) -> term()

连续调用 get_value/2 函数来处理 Key 剩下的部分，Key 中子 key 用 “.” 分, 用上一级 get_value 返回的值替换 Args . 用于执行格式如 {0.0} 的情况.

get_field/3

get_field(Key::string(), Args, Module) -> term()

同上.

get_value/2

get_value(Key::string(), Args::args()) -> term()

从 args 中获取 key 的值.如果 args 是一个元组，然后会把 key 转成一个整数，然后通过(1+key, args)来获取元素，并返回. 如果 Args 是一个列表，并且 key 可以转化成一个整数，那么将返回 lists:nth(1 + Key, Args), otherwise try and look for Key in Args as a proplist, converting Key to an atom or binary if necessary.

tokenize/1

tokenize(S::string()) -> tokens()

将字符串格式化成 mochiweb 的网络格式.
